4. 格式化输入函数 scanf()

  • 一般调用格式为 > scanf( 格式控制符 , 地址列表 )

  • 地址列表中给出各变量的地址，可以是变量的地址，也可以是字符串的首地址

  • 格式控制符由%和格式符组成，作用是将要输入的字符按指定格式输入

scanf 函数的格式符

格式符 说明
d,i 用于输入十进制整数
u 以无符号十进制形式输入十进制整数
o(字母) 输入八进制整数
x 输入十六进制整数
c 用于输入单个字符
s 用于输入字符串（非空格开始，空格结束，字符串变量以'\0'结尾）
f,e 用于输入实数（小数与指数皆可）
l 用于输入长整型数(%ld、%lo、%lx)或double型实数(%lf、%le)
h 用于短整型数(%hd、%ho%、%hx)

注：

  • 不要忘记寻址！不要忘记寻址！不要忘记寻址！

  • scanf()实际的输入方式是读取字符，格式符用于转换

  • "%d"的意思是将字符改为十进制数，其只识别十进制数，空格、回车和Tab会被当做间隔符，但是"%c"不是！

  • "%c"会录入所有的字符，包括空格、回车和Tab，它们在此时不被视为间隔符!



5. 格式化输出函数 print()

  • 一般调用格式为 > print( 格式控制符 , 地址列表 )

printf()函数的格式符

格式符 说明
d，i 以带符号的十进制形式输出整数，正数的+号不显示
u 以无符号的十进制输出整数
x (或X) 以十六进制无符号形式输出整数（不输出前导符0x）
o 以八进制无符号形式输出整数
c 输出一个字符
s 输出字符串
f 以小数形式输出单、双精度，隐含输出6位小数
e (或E) 以指数形式输出单、双精度，隐含输出6位小数

d格式符

参数 说明
%d 输出数字长为变量数值的实际长度
%md 输出m位（不足时补空格，大运m位是按实际长度输出）
%-md m含义同上。左对齐1输出
%0md m位，不够补0

f格式符

参数 说明
%f 按实数格式输出，整数部分按实际位数输出，6位小数
%m.nf 总位数m(含小数点)，其中有n位小数
%-m.nf 同上，左对齐2

s格式符

参数 说明
%s 按实际宽度输出一个字符串
%ms 制定宽度，不足时左补空格，大于时按实际宽度
%-ms 左对齐3，同上
%m.ns 输出占m个字符位置，其中字符数最多n个，左补空格
%-m.ns 左对齐4，同上



6. 几种格式的说明

  • cin与cout在 Dev C++ 中只能调用<iostream>库 ，而其他输入输出格式要调用<stdio.h>库或者<cstdio>库。
  • cin与cout属于C++的概念，而 scanf 和 printf 属于 C 的概念，在时效上 scanf 与 printf 更优
  • string 属于C++的概念，因此不能用 printf 和 scanf ，如果需要 printf 中 可以用 .c_str() 函数返回字符串首字母的地址。
